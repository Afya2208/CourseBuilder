# CourseBuilder Website - веб-сайт конструктор курсов

Это веб-сайт, на котором можно создавать образовательные курсы.
Пользователи делятся на 3 группы: администраторы, разработчики и гости (неавторизованные пользователи). 
Гости могут только смотреть курсы, разработчики могут их создавать и редактировать, а администраторы управляют аккаунтами разработчиков и администраторов

Веб-интерфейс разработан на Vue.js 3 на TypeScript + Vite

Дизайн сделан с помощью Bootstrap 5 + BootstrapVueNext
